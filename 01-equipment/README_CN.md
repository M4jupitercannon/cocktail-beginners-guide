## 必备工具

### 1. 量酒器 (Jigger)

量酒器是调酒中最重要的工具之一。精确量取每种配料是实现鸡尾酒平衡口感的关键。

- **类型**：双头量酒器（常见 30ml/45ml 或 25ml/50ml 组合）、刻度量杯
- **推荐**：带有内刻度线的量酒器，可以量取 15ml、22.5ml、30ml、45ml 等多种容量
- **使用技巧**：倒酒时保持量酒器水平，液面与杯口齐平即为标准量
- **注意**：不同品牌的量酒器精度差异很大，购买后建议用厨房秤校验（1ml 水 = 1g）

### 2. 调酒器 (Cocktail Shaker)

调酒器用于摇酒（Shaking），是混合、冰镇和稀释鸡尾酒的核心工具。

#### 三件式调酒器 (Cobbler Shaker)
- **组成**：底杯 + 内置滤网中段 + 顶盖
- **优点**：自带滤网，适合初学者；不需额外购买滤冰器
- **缺点**：出液较慢；顶盖冰冻后可能卡住难以打开
- **适合**：初学者、家用

#### 两件式调酒器 (Boston / French Shaker)
- **组成**：大锡杯 + 小锡杯（French）或大锡杯 + 玻璃杯（Boston）
- **优点**：出液快，容量大，易清洗，适合专业使用
- **缺点**：需要搭配 Hawthorne 滤冰器；需要一定练习掌握开合技巧
- **推荐**：锡对锡（French Shaker）优于玻璃对锡（Boston Shaker）
- **适合**：希望进阶的爱好者、专业调酒师

### 3. 吧匙 (Bar Spoon)

吧匙用于搅拌鸡尾酒，也可用于分层、量取少量液体。

- **长度**：至少 30cm，理想为 30-34cm
- **类型**：螺旋柄（最经典）、直柄、叉形末端、扁平圆盘末端
- **推荐**：带螺旋柄和扁平圆盘末端的款式，方便分层倒酒
- **勺容量**：标准吧匙约 5ml（1 茶匙）
- **使用**：搅拌时吧匙沿杯壁内侧滑动，用手指转动而非手腕

### 4. 滤冰器 (Strainer)

#### Hawthorne 滤冰器
- **用途**：搭配两件式调酒器使用，过滤摇酒后的冰块和碎片
- **特征**：带弹簧环，可以卡在调酒器口部
- **使用**：按压手指支撑点可以调节过滤精度

#### Julep 滤冰器
- **用途**：搭配搅拌杯使用，过滤搅拌后的鸡尾酒
- **特征**：穿孔金属勺，略小于搅拌杯口径

#### 细滤网 (Fine Strainer)
- **用途**：双重过滤（Double Straining），去除细小冰渣和果肉碎屑
- **使用**：放在杯口上方，将摇好的鸡尾酒通过 Hawthorne 滤冰器和细滤网双重过滤

### 5. 捣棒 (Muddler)

- **用途**：捣碎水果、香草、香料以释放风味
- **材质**：木质（传统但不耐水泡）、不锈钢/塑料组合（耐用）
- **技巧**：用捣棒按压并旋转，轻柔地捣碎。对薄荷只需轻拍释放精油，过度捣碎会产生苦味
- **安全**：只在不锈钢调酒器底部或坚固的玻璃杯中捣碎，切勿在薄玻璃杯中用力捣

### 6. 榨汁器 (Citrus Juicer)

- **类型**：手持铰链式（"墨西哥肘"）、杠杆式立式榨汁器、电动榨汁器
- **推荐初学者**：手持铰链式，分为青柠（绿色）、柠檬（黄色）、橙子（橙色）三种尺寸
- **技巧**：榨汁后通过细滤网过滤掉果肉和籽
- **重要原则**：**永远使用新鲜榨取的柑橘汁**，这是好鸡尾酒的灵魂

## 进阶工具

| 工具 | 用途 | 必要性 |
|------|------|--------|
| **搅拌杯 (Mixing Glass)** | 专用于搅拌式鸡尾酒，比用调酒器搅拌更优雅 | 推荐 |
| **削皮刀 (Peeler)** | 切取柑橘皮装饰（Twist） | 推荐 |
| **小刀 (Paring Knife)** | 切割水果和装饰物 | 推荐 |
| **砧板 (Cutting Board)** | 切割水果和装饰物的工作台 | 推荐 |
| **冰格/冰模 (Ice Tray)** | 制作大方冰块、圆冰球 | 推荐 |
| **开瓶器 (Corkscrew)** | 打开葡萄酒和苦艾酒瓶 | 推荐 |
| **苦精瓶 (Dasher Bottle)** | 精确控制苦精滴量 | 可选 |
| **喷雾瓶 (Atomizer)** | 喷洒苦艾酒、苦精等 | 可选 |
| **肉豆蔻刨 (Grater)** | 现磨肉豆蔻粉装饰 | 可选 |
| **搅拌机 (Blender)** | 冰沙类鸡尾酒（Frozen Daiquiri 等） | 可选 |
| **碎冰袋 (Lewis Bag)** | 用木槌敲碎冰块 | 可选 |

## 杯具指南

### 核心杯型

#### 1. 碟形杯 / 轿跑杯 (Coupe Glass)
- **容量**：150-180ml
- **用途**：经典纯饮鸡尾酒（Daiquiri、Sidecar、Gimlet）
- **特点**：宽口浅碗，现代调酒中逐渐取代马天尼杯
- **持杯**：握杯柄，避免手温影响酒液温度

#### 2. 马天尼杯 (Martini Glass / Cocktail Glass)
- **容量**：150-200ml
- **用途**：Martini、Cosmopolitan、Manhattan
- **特点**：V 形锥体，标志性外观
- **持杯**：握杯柄或杯底

#### 3. 古典杯 / 岩石杯 (Rocks Glass / Old Fashioned Glass)
- **容量**：200-300ml
- **用途**：加冰短饮（Old Fashioned、Negroni、Whiskey Sour on rocks）
- **特点**：矮壮、宽口、厚底
- **变体**：双份古典杯（Double Old Fashioned），约 350-400ml

#### 4. 高球杯 (Highball Glass)
- **容量**：240-350ml
- **用途**：长饮类鸡尾酒（Gin & Tonic、Highball、Moscow Mule）
- **特点**：直筒高杯

#### 5. 柯林斯杯 (Collins Glass)
- **容量**：300-400ml
- **用途**：Tom Collins、Mojito、长饮
- **特点**：比高球杯更高更窄

#### 6. 铜杯 (Copper Mug)
- **容量**：350-500ml
- **用途**：Moscow Mule
- **特点**：铜制，保冷效果好，标志性外观

## 初学者采购建议

### 第一阶段：最低预算启动包

如果你预算有限，以下是最基本的起步套装：

1. **三件式调酒器** x1（自带滤网，省掉单独买滤冰器）
2. **量酒器** x1（带刻度的双头款）
3. **吧匙** x1
4. **手持榨汁器** x1
5. **古典杯** x2 + **高球杯** x2

> 预算参考：约 100-200 元即可配齐

### 第二阶段：进阶升级

在第一阶段基础上增加：

6. **两件式调酒器 (French Shaker)** x1
7. **Hawthorne 滤冰器** x1
8. **细滤网** x1
9. **搅拌杯** x1
10. **碟形杯 (Coupe)** x2
11. **削皮刀** x1
12. **大方冰模** x1

> 预算参考：额外 200-400 元

### 购买渠道建议

- **线上**：淘宝/京东搜索"调酒套装"、"bartender kit"
- **推荐品牌**：Cocktail Kingdom（专业级）、IKEA（基础杯具性价比高）
- **注意事项**：
  - 不要买太便宜的调酒器，薄壁不锈钢手感差且隔热效果不好
  - 量酒器优先选择有内刻度线的款式
  - 杯具先买基础款，不必追求花哨
