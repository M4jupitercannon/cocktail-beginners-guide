# 调酒术语表（中英对照）

> 调酒世界有许多专业术语。掌握这些术语能帮助你更好地理解配方和教学资料。

[English Version](en.md)

---

## 基础调酒术语

| 英文 | 中文 | 解释 |
|------|------|------|
| **ABV** (Alcohol By Volume) | 酒精度 | 酒精含量百分比，如 40% ABV |
| **Aperitif** | 餐前酒 | 餐前饮用的低酒精度饮品，刺激食欲 |
| **Back** | 配饮 | 与鸡尾酒搭配的饮料，如一杯水或啤酒 |
| **Bitters** | 苦精 | 高浓度植物风味提取液，按"滴"使用 |
| **Blend** | 搅拌机混合 | 用搅拌机将配料和冰一起打碎 |
| **Build** | 直接注入 | 直接在酒杯中混合配料 |
| **Call drink** | 指名酒款 | 客人指定品牌的点单方式，如"Tanqueray & Tonic" |
| **Chaser** | 追饮 | 喝烈酒后紧跟的饮料，缓解刺激 |
| **Cobbler** | 三件式调酒器 | 带内置滤网的调酒器 |
| **Cocktail** | 鸡尾酒 | 混合两种或以上配料制成的饮品 |
| **Cordial** | 甜利口酒 / 浓缩糖浆 | 在美国指利口酒，在英国指浓缩果汁糖浆 |
| **Dash** | 一甩 / 一滴 | 苦精的计量单位，约 0.8-1ml |
| **Digestif** | 餐后酒 | 餐后饮用的酒，助消化 |
| **Dirty** | 脏的 | 加入橄榄汁（如 Dirty Martini） |
| **Double strain** | 双重过滤 | 同时使用 Hawthorne 滤冰器和细滤网 |
| **Dry** | 干的 | 不甜的，或少量使用苦艾酒 |
| **Dry shake** | 干摇 | 不加冰摇晃，用于乳化蛋清 |
| **Express** | 挤压（柑橘皮） | 挤压柑橘皮释放精油 |
| **Fine strain** | 精细过滤 | 用细网滤去冰渣和碎屑 |
| **Float** | 浮层 | 将液体轻轻浮在另一种液体表面 |
| **Free pour** | 自由倒酒 | 不使用量酒器，凭感觉倒酒（不推荐初学者） |
| **Garnish** | 装饰物 | 用于装饰和增添风味的水果、香草等 |
| **Highball** | 高球 / 长饮 | 基酒 + 碳酸饮料的简单长饮 |
| **Jigger** | 量酒器 | 用于精确量取液体配料 |
| **Layer** | 分层 | 利用比重差异将液体叠加 |
| **Long drink** | 长饮 | 容量较大，含碳酸或果汁的鸡尾酒 |
| **Mixology** | 调酒学 | 鸡尾酒创作和调制的艺术与科学 |
| **Muddle** | 捣碎 | 用捣棒按压释放风味 |
| **Muddler** | 捣棒 | 用于捣碎水果和香草的工具 |
| **Neat** | 纯饮 | 不加冰、不加水、不加任何东西地直接饮用 |
| **On the rocks** | 加冰 | 将酒倒在冰块上 |
| **Proof** | 酒度 | 美制酒精度量，1 proof = 0.5% ABV（80 proof = 40% ABV） |
| **Rim** | 杯沿 / 沾边 | 在杯口涂抹盐或糖 |
| **Rinse** | 涮杯 | 用少量烈酒涮洗杯壁（如苦艾酒涮杯） |
| **Roll** | 滚动 | 在两个容器之间来回倒 |
| **Shake** | 摇酒 | 将配料和冰在调酒器中摇晃 |
| **Short drink** | 短饮 | 容量较小、酒精浓度较高的鸡尾酒 |
| **Simple syrup** | 简易糖浆 | 1:1 的糖水溶液 |
| **Sour** | 酸酒 | 基酒 + 柑橘汁 + 甜味剂的鸡尾酒家族 |
| **Stir** | 搅拌 | 用吧匙在搅拌杯中搅拌 |
| **Straight up / Up** | 纯饮（冰镇后不加冰） | 摇/搅后滤入不加冰的杯中 |
| **Strain** | 过滤 | 将液体与冰块分离 |
| **Swizzle** | 搅棒搅拌 | 用搅棒在碎冰中旋转搅拌 |
| **Throw** | 掷酒 | 在两个容器之间远距离抛倒 |
| **Top / Top up** | 补满 | 用碳酸饮料或其他液体补满杯子 |
| **Twist** | 柑橘皮卷 | 扭转的柑橘皮装饰 |
| **Virgin / Mocktail** | 无酒精版 / 模拟鸡尾酒 | 不含酒精的鸡尾酒 |
| **Wash line** | 液面线 | 倒好的鸡尾酒在杯中的液面高度 |
| **Well drink** | 廉价酒款 | 使用吧台标配的普通品牌基酒 |
| **Wet** | 湿的 | 多加苦艾酒（如 Wet Martini） |
| **Zest** | 柑橘皮屑 | 柑橘表皮的细小碎屑 |

## 酒类术语

| 英文 | 中文 | 解释 |
|------|------|------|
| **Agave** | 龙舌兰（植物） | 制作龙舌兰酒的原料植物 |
| **Aged / Añejo** | 陈年 | 在橡木桶中陈化 |
| **Blanco / Silver** | 白色/银色 | 未陈年的龙舌兰酒 |
| **Blended** | 调和 | 多种酒混合调配 |
| **Botanicals** | 植物原料 | 金酒中使用的香料和植物（杜松子等） |
| **Bourbon** | 波本 | 美国威士忌，至少 51% 玉米 |
| **Brandy** | 白兰地 | 水果（通常是葡萄）蒸馏酒 |
| **Cask strength** | 桶强 | 未稀释的原桶酒精度 |
| **Cognac** | 干邑 | 法国干邑地区产的白兰地 |
| **Curaçao** | 库拉索 | 以苦橙皮制作的利口酒 |
| **Genever** | 荷兰金酒 | 金酒的荷兰前身 |
| **Grain spirit** | 谷物烈酒 | 以谷物为原料蒸馏的酒 |
| **Juniper** | 杜松子 | 金酒的核心风味原料 |
| **Liqueur** | 利口酒 | 加糖加味的烈酒 |
| **London Dry** | 伦敦干金酒 | 金酒的经典风格 |
| **Malt** | 麦芽 | 发芽的大麦，威士忌原料 |
| **Mezcal** | 梅斯卡尔 | 墨西哥龙舌兰蒸馏酒（龙舌兰酒的"兄弟"） |
| **Molasses** | 糖蜜 | 制作朗姆酒的原料之一 |
| **Overproof** | 超标准酒精度 | 酒精度超过 50% ABV 的烈酒 |
| **Reposado** | 微陈年 | 陈年 2-12 个月的龙舌兰酒 |
| **Rye** | 黑麦（威士忌） | 以黑麦为主要原料的威士忌 |
| **Single malt** | 单一麦芽 | 来自单一蒸馏厂的麦芽威士忌 |
| **Triple sec** | 三重蒸馏橙味利口酒 | 橙味利口酒的通称 |
| **Vermouth** | 苦艾酒 / 味美思 | 加香的加强葡萄酒 |

## 计量单位

| 单位 | 换算 |
|------|------|
| 1 oz (盎司) | ≈ 30ml |
| 1.5 oz | ≈ 45ml |
| 2 oz | ≈ 60ml |
| 1 dash | ≈ 0.8-1ml |
| 1 bar spoon | ≈ 5ml (1 茶匙) |
| 1 splash | ≈ 5-10ml |
| 1 pony | = 1 oz = 30ml |
| 1 jigger | = 1.5 oz = 45ml |
