## Learning Path / 学习路径

```
Week 1 / 第1周:  购买基本装备，练习摇酒和搅拌手法
        ↓
Week 2-3 / 第2-3周:  从最简单的4款鸡尾酒开始 (G&T, Moscow Mule, Highball, Paloma)
        ↓
Week 4-5 / 第4-5周:  学做 Daiquiri、Whiskey Sour、Margarita — 掌握摇酒技术
        ↓
Week 6-7 / 第6-7周:  尝试 Old Fashioned、Negroni、Manhattan — 掌握搅拌技术
        ↓
Week 8+ / 第8周起:   挑战进阶鸡尾酒，尝试变体配方，创造自己的风格
```
