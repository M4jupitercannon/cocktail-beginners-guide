## 1. 五味基础

鸡尾酒的风味主要由以下五种味觉元素构成：

| 味觉 | 来源 | 作用 |
|------|------|------|
| **甜 (Sweet)** | 糖浆、利口酒、甜苦艾酒、果汁 | 增加圆润感和饱满度，软化酒精的刺激 |
| **酸 (Sour)** | 柠檬汁、青柠汁、葡萄柚汁 | 增加清爽感和活力，平衡甜味 |
| **苦 (Bitter)** | 苦精、金巴利、苦艾酒、龙胆草 | 增加复杂度和深度，刺激食欲 |
| **咸 (Salt)** | 盐边、盐水溶液 | 少量盐能增强其他风味（类似烹饪中的作用） |
| **鲜/辣 (Umami/Spice)** | 辣椒、胡椒、姜、香料 | 增加刺激感和独特个性 |

### 第六味：酒精本身

酒精不属于传统五味，但它带来**温热感**和**灼烧感**，是鸡尾酒的"骨架"。酒精含量过高会让鸡尾酒过于刺激，过低则显得单薄。

## 2. 鸡尾酒的平衡法则

### 核心原则：酸甜平衡

绝大多数鸡尾酒的核心是**酸与甜的平衡**。就像做菜一样，太甜会腻，太酸会刺。

**基本比例参考**（以 Sour 类鸡尾酒为例）：

```
基酒 : 酸味 : 甜味 = 2 : 1 : 0.75 (经典比例)

例如 Daiquiri:
60ml 朗姆酒 : 25ml 青柠汁 : 15ml 糖浆
```

### 调整技巧

| 问题 | 解决方法 |
|------|---------|
| 太甜 | 加一点酸（柠檬/青柠汁） |
| 太酸 | 加一点甜（糖浆） |
| 太烈/太辣 | 增加稀释（多摇几秒）或加更多甜/酸 |
| 太淡 | 减少稀释或增加基酒比例 |
| 风味平淡 | 加 1-2 滴苦精，或少量盐 |

### 稀释的重要性

一杯好鸡尾酒通常含 **15-25%** 的水（来自冰的融化）。适量的稀释能：
- 降低酒精刺激感
- 让各种风味更好地融合
- 使鸡尾酒变得更"顺滑可饮"

这就是为什么摇酒和搅拌的时间很重要——它决定了稀释程度。

## 3. 鸡尾酒家族分类

大多数鸡尾酒可以归入几个"家族"。理解这些家族的结构，你就能举一反三地理解上千种鸡尾酒。

### Sour 家族（酸类）

**公式**：基酒 + 柑橘汁（酸）+ 甜味剂

| 变体 | 公式 | 代表 |
|------|------|------|
| **基本 Sour** | 基酒 + 柠檬/青柠 + 糖浆 | Daiquiri, Whiskey Sour, Gimlet |
| **加利口酒的 Sour** | 基酒 + 柑橘 + 利口酒（代替糖浆） | Margarita (龙舌兰+青柠+Cointreau), Sidecar |
| **加蛋清的 Sour** | 基酒 + 柑橘 + 糖浆 + 蛋清 | Pisco Sour, Clover Club |
| **Collins** | Sour + 苏打水 (长饮版) | Tom Collins, John Collins |
| **Fizz** | Sour + 苏打水 (短饮版) | Gin Fizz, Ramos Gin Fizz |

### Old Fashioned 家族（古典类）

**公式**：基酒 + 糖 + 苦精

| 变体 | 代表 |
|------|------|
| 波本/黑麦 + 糖 + Angostura | Old Fashioned |
| 干邑 + 糖 + Peychaud's + 苦艾酒 | Sazerac |
| 龙舌兰 + 龙舌兰蜜 + 苦精 | Oaxaca Old Fashioned |

### Martini 家族（马天尼类）

**公式**：基酒 + 苦艾酒（或其他配制酒）

| 变体 | 代表 |
|------|------|
| 金酒 + 干苦艾酒 | Dry Martini |
| 伏特加 + 干苦艾酒 | Vodka Martini |
| 金酒 + 甜苦艾酒 + 金巴利 | Negroni |
| 黑麦 + 甜苦艾酒 + 苦精 | Manhattan |

### Highball 家族（长饮类）

**公式**：基酒 + 碳酸/非碳酸长饮

| 变体 | 代表 |
|------|------|
| 威士忌 + 苏打水 | Highball |
| 金酒 + 汤力水 | Gin & Tonic |
| 伏特加 + 姜汁啤酒 + 青柠 | Moscow Mule |
| 龙舌兰 + 葡萄柚汽水 | Paloma |
| 朗姆 + 可乐 | Cuba Libre |

### Tiki 家族（热带类）

**公式**：朗姆（通常多种混合）+ 柑橘 + 糖浆/利口酒 + 异域辅料

| 代表 | 特点 |
|------|------|
| Mai Tai | 双朗姆 + 青柠 + 橙味利口酒 + 杏仁糖浆 |
| Jungle Bird | 黑朗姆 + 金巴利 + 菠萝汁 + 青柠 |
| Piña Colada | 朗姆 + 椰奶 + 菠萝汁 |

## 4. 配方设计思路

当你想创造自己的鸡尾酒时，可以遵循以下步骤：

### 步骤 1：选择一个家族模板

从上面的家族中选一个你喜欢的结构作为起点。

### 步骤 2：选择基酒

根据你想要的风味方向选择基酒。例如：
- 想要干净清爽 → 伏特加或白朗姆
- 想要植物芳香 → 金酒
- 想要温暖醇厚 → 波本或黑麦
- 想要辛辣独特 → 龙舌兰

### 步骤 3：添加修饰

用利口酒、苦精、糖浆等增加复杂度和个性。

### 步骤 4：平衡调整

制作一杯试饮，然后根据五味原则调整：
- 太甜？加酸。
- 太酸？加甜。
- 太简单？加一滴苦精或少量盐。

### 步骤 5：命名和装饰

为你的创作取一个名字，选择合适的杯具和装饰。

## 5. 风味轮与搭配参考

### 经典风味搭配

| 基酒 | 最佳搭档 |
|------|---------|
| **伏特加** | 几乎任何风味；柑橘、浆果、咖啡、姜 |
| **金酒** | 柑橘、黄瓜、薰衣草、迷迭香、汤力水 |
| **白朗姆** | 青柠、薄荷、椰子、菠萝、百香果 |
| **龙舌兰** | 青柠、葡萄柚、芒果、辣椒、龙舌兰蜜 |
| **波本** | 蜂蜜、柠檬、樱桃、苹果、肉桂 |
| **黑麦** | 樱桃、苦精、苦艾酒、橙皮 |

### 季节性搭配建议

| 季节 | 推荐方向 | 代表鸡尾酒 |
|------|---------|-----------|
| **春** | 花香、清新、草本 | Gimlet, Tom Collins, Gin Basil Smash |
| **夏** | 热带、冰爽、水果 | Mojito, Daiquiri, Paloma |
| **秋** | 温暖、香料、苹果 | Old Fashioned, Manhattan, Apple Cider Cocktails |
| **冬** | 浓郁、咖啡、巧克力 | Espresso Martini, Hot Toddy, Irish Coffee |
