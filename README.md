## Learning Path / 学习路径

```
Week 1 / 第1周:  Purchase basic equipment, practice shaking & stirring
                  购买基本装备，练习摇酒和搅拌手法
        ↓
Week 2-3 / 第2-3周:  Start with 4 easiest cocktails (G&T, Moscow Mule, Highball, Paloma)
                      从最简单的4款鸡尾酒开始
        ↓
Week 4-5 / 第4-5周:  Learn Daiquiri, Whiskey Sour, Margarita — master shaking
                      学做代基里、威士忌酸、玛格丽特 — 掌握摇酒技术
        ↓
Week 6-7 / 第6-7周:  Try Old Fashioned, Negroni, Manhattan — master stirring
                      尝试古典、内格罗尼、曼哈顿 — 掌握搅拌技术
        ↓
Week 8+ / 第8周起:   Challenge advanced cocktails, experiment with variations
                      挑战进阶鸡尾酒，尝试变体配方，创造自己的风格
```
